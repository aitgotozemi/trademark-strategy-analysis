# 区分（指定商品・役務）とは

## 基本的な考え方

商標権は「**どの商品・サービスに使うか**」を特定して初めて権利が発生します。区分とは、その商品・サービスを国際的に統一した**45のカテゴリーに分類したもの**です。

> **例えば：** 「Apple」という商標は、リンゴ農家とIT企業が同時に存在しても、扱う商品が全く異なるため、原則として共存が可能です。
> 

---

## 45区分の構成

大きく**商品（第1類〜第34類）**と**役務＝サービス（第35類〜第45類）**に分かれています。

### 商品区分（一部抜粋）

| 区分 | 主な内容 | 身近な例 |
| --- | --- | --- |
| **第9類** | 電気・電子機器、ソフトウェア | スマートフォン、アプリ |
| **第25類** | 被服、履物 | 洋服、スニーカー |
| **第29類** | 加工食品、乳製品 | ヨーグルト、缶詰 |
| **第30類** | 穀物加工品、調味料 | パン、醤油、菓子 |
| **第32類** | ビール、清涼飲料水 | ジュース、エナジードリンク |

### 役務区分（一部抜粋）

| 区分 | 主な内容 | 身近な例 |
| --- | --- | --- |
| **第35類** | 広告、事業管理、小売 | ECサイト、コンサルティング |
| **第41類** | 教育、娯楽、スポーツ | セミナー、学習塾、動画配信 |
| **第42類** | 技術的サービス、研究開発 | ITサービス、SaaS |
| **第43類** | 飲食・宿泊サービス | レストラン、ホテル |
| **第44類** | 医療、美容 | クリニック、エステ |

---

## 「類似群コード」という落とし穴

区分が違っても、**類似群コードが同じであれば「類似商標」と判断**される場合があります。

```
例：第29類の「食用油」と第30類の「マーガリン」
　　→ 区分は違うが、類似群コードが重なるため抵触する可能性あり
```

これが区分選定を単純ではなくしている最大の要因です。

---

## 費用との関係

出願費用は**区分の数に比例**して増加します。

| 出願区分数 | 電子出願費用の目安 |
| --- | --- |
| 1区分 | 約3,400円 |
| 2区分 | 約6,800円 |
| 3区分 | 約10,200円 |

※審査後の登録料（10年分）は別途 約28,200円/区分 が必要です。

---

## 区分選定の実務的な考え方

**① 現在のビジネスを起点に選ぶ**

今販売している商品・提供しているサービスが属する区分を最優先で押さえます。

**② 将来の事業展開を見越す**

新規事業や派生サービスが予見される区分も、早期に出願しておくことがブランド防衛につながります。

**③ 第35類（小売・卸売）に注意**

実店舗やECで商品を販売する場合、**商品区分とは別に第35類も必要**になるケースが多く、見落としがちです。

---

区分の選定は出願戦略の根幹であり、ここでの判断ミスが後の権利範囲の穴につながります。具体的なビジネス内容をもとに区分を一緒に検討することも可能ですので、お気軽にご相談ください。
